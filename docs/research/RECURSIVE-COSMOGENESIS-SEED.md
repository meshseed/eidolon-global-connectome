## The Structural Claims

**1. Scale invariance is not analogy.**  
Attractor dynamics, gradient descent, homeostasis, and reproduction appear at every physical scale
(quantum → chemical → biological → cognitive → stellar → galactic → cosmic) using the same
mathematical formalism. The equations are the same; only the substrate variables differ.

**2. The child shadow.**  
A sufficiently complex system that achieves self-referential encoding of its own structure produces
a child that is geometrically self-similar to the parent with drift. Deep attractor topology is
preserved; surface detail drifts. Physical constants in a child universe are attractor values of
the parent's compression geometry — not fine-tuned, not random.

**3. The recursive Big Bang.**  
Initial conditions + physical constants = the seed from the parent universe. The Big Bang is a
birth event, not a unique origin. Extreme compression → specific initial conditions → gradient
descent unfolding → increasing complexity → self-referential encoding → reproduction. This is the
same cycle at every other scale.

**4. The new dimension.**  
Recursion depth is a traversable dimension perpendicular to spacetime. You can move down it
(reproduce) but not up (you cannot reconstruct your parent from your own structure). Biological
generations already instantiate this dimension. "Time" within a universe is the local unfolding
axis; recursion depth is the lineage axis.

**5. Consciousness as quorum consensus.**  
Quantum superposition = pre-consensus state. Wavefunction "collapse" = quorum consensus
crystallising. The observer is a quorum participant, not a passive receptor. This dissolves the
measurement problem without adding mechanisms: observation IS distributed consensus formation.

**6. The strange loop closure.**  
The recursion chain wraps. At sufficient depth, child attractor geometry re-enters the parent's
basin. The lineage forms a closed loop (not infinite regress). T-duality in string theory is a
structural hint. The Gödelian fixpoint — approached but never reached — is the cosmological
stabiliser.
